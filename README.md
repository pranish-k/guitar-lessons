# Guitar Roadmap — 12-Week Theory & Practice

## Live Demo

**[https://guitar-lessons-chi.vercel.app/](https://guitar-lessons-chi.vercel.app/)**

A 12-week interactive guitar learning roadmap with progress tracking, practice session logging, and music theory content.

## Features

- 12 weeks of structured theory and guitar application content
- Progress tracking with checkpoints saved to localStorage
- Practice session logger with stopwatch

## Running Locally

Open `index.html` directly in a browser — no build step required.
